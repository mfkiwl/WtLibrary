# 摘要
一种超精简的事件驱动型有限状态机(Mealy)，使用指针函数实现，非常适用于资源有限的和低功耗场景。编码遵循 ANSI C 规范，易于扩展和学习。
其工作过程如下：
- 使用指定起始状态和最终状态初始化状态机，并设置状态机的当前状态为起始状态，状态机开始工作。
- 在相关事件发生时，把事件关联的参数传递给状态机并执行状态转换活动。
- 如果状态机进入最终状态（使用当前状态是否等于最终状态来判断），则状态机停机；否则继续工作。
# 注意事项
- 部分开发环境需要预先初始化动态内存池作为使用动态内存的前置条件(如 Keil C51)。
- 状态机不需要工作时，一定要调用函数 FsmTiny_Dispose 释放占用资源，否则有可能造成内存泄露。

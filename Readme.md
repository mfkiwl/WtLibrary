# 摘要

C 语言应用级扩展库。是在工作、学习过程中积累或研发的应用模块集合。编码遵循 
ANSI C 规范，易于扩展和学习。
常用模块：
- 状态机：FsmTiny
- 按键：WtButton
- 灯控：WtLamp
